## 如何使用

1. 安装node.js

2. `git clone git@github.com:KveinAxel/NewsBlockChainFrontEnd.git`

3. `cd NewsBlockChainFrontEnd\`

4. `npm install`

5. 配置 `/NewsBlockChainFrontEnd/config/` 下的url

5. `npm run dev`
